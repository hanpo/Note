# iPhone Camera

2011/12/25 by Hanpo 

###------iPhone相機整理------

####iPhone 3G
	螢幕	 	480 x 320
	拍照		1600 x 1200
	錄影		無
	備註		只能按快門的簡單相機

####iPhone 3GS
	螢幕	 	480 x 320
	拍照		2048 x 1536
	錄影		640 x 480（30fps）
	備註		自動對焦、點擊對焦、無縫切換近拍模式(Macro Mode)，自動白平衡

####iPod touch 4th
	螢幕	 	960 x 640
	拍照		960 x 720
	錄影		720p(30fps)
	備註		前置VGA視訊相機

####iPhone 4
	螢幕	 	960 x 640
	拍照		2592 x 1936
	錄影		720p(30fps)
	備註		LED閃光燈功能、前置VGA視訊相機

####iPhone 4s
	螢幕	 	960 x 640
	拍照		3264 x 2448
	錄影		1080p(30fps)
	備註		LED閃光燈功能、前置VGA視訊相機
